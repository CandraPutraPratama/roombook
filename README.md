# RoomBook - Sistem Reservasi Ruangan Kampus

RoomBook adalah aplikasi web sederhana untuk mengelola reservasi ruangan kampus. Aplikasi ini dibuat menggunakan PHP Native dan PostgreSQL.

## Fitur Utama

- Dashboard ringkasan data
- Manajemen data ruangan
- Manajemen reservasi ruangan
- Validasi jadwal bentrok
- Jadwal penggunaan ruangan
- Riwayat reservasi
- Search dan filter data
- UI responsive

## Teknologi

- PHP Native
- PostgreSQL
- HTML
- CSS
- Bootstrap
- JavaScript
